Voltage Derate Limit to Zero Timer

Units: [100ms]

This parameter is the number of 100ms intervals that must pass (when the voltage limit is exceeded) before the associated battery current limit is cut by 10% of the maximum limit. The timer allows the option of having brief pulses above the maximum cell voltage for specific applications (EG: allowing very brief spikes when a generator turns on).

A value of 0 means that the associated limit (CCL / DCL) will immediately derate to 0 when a cell voltage exceeds the maximum voltage setting.
