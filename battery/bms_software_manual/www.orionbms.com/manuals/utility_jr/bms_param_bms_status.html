BMS Status

Default Length (Bytes): 2

This parameter contains the status of several BMS operation values. The format is as follows:

Bit #1 (0x0001): Voltage Failsafe Status
Bit #2 (0x0002): Current Failsafe Status
Bit #3 (0x0004): Relay Failsafe Status
Bit #3 (0x0008): Cell Balancing Active

For all of these values, a "1' signifies that the signal or relay in question is enabled or active (high). A "0' indicates that it is inactive or off (low).
