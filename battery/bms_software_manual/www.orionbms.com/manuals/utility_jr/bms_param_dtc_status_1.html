DTC Status #1

Default Length (Bytes): 2

This parameter contains the status of several BMS fault codes. The format is as follows:

Bit #1 (0x0001): P0A1F (Internal Communication Fault)
Bit #2 (0x0002): P0A00 (Internal Conversion Fault)
Bit #3 (0x0004): P0A80 (Weak Cell Fault)
Bit #4 (0x0008): P0AFA (Low Cell Voltage Fault)
Bit #5 (0x0010): P0A04 (Open Wiring Fault)
Bit #6 (0x0020): P0AC0 (Current Sensor Fault)
Bit #7 (0x0040): P0A01 (Pack Voltage Sensor Fault)
Bit #8 (0x0080): P0A03 (Voltage Redundancy Fault)

Bit #9 (0x0100): P0A02 (Weak Pack Fault)
Bit #10 (0x0200): RESERVED
Bit #11 (0x0400): P0A9C (Thermistor Fault)
Bit #12 (0x0800): RESERVED
Bit #13 (0x1000): RESERVED
Bit #14 (0x2000): P0A0D (Cell Voltage Over 5V)
Bit #15 (0x4000): RESERVED
Bit #16 (0x8000): P0A06 (Charge Limit Enforcement Fault)

For all of these values, a "1' signifies that the signal or relay in question is enabled or active (high). A "0' indicates that it is inactive or off (low).
