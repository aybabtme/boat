Discharged SOC

Units: [SOC%]

The State of Charge (SOC)% at which the pack is deemed to be fully discharged. This setting is used to compute total Pack Health by comparing a relationship between observed capacity and expected capacity. It should be set to a SOC value that the pack will fairly regularly be discharged to so the BMS can perform a capacity health check regularly.

NOTE: This value has no affect on when the BMS turns off discharge. It is ONLY used for pack health calculations.
