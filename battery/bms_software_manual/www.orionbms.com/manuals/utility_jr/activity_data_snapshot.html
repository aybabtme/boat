Taking a Data Snapshot From The BMS

Taking a data snapshot is a very good way to capture all the current (and stored diagnostic) information from the BMS and collect it all into a single archive. A data snapshot may be requested for diagnosing and resolving technical support questions and issues.

  1. Open the OrionBMS JR utility.

  2. Connect to the BMS.

  3. Go to the Help menu item at the top

  4. Select the "Take Data Snapshot" item from the menu.

  5. Specify a location on the computer where to save the data snapshot (it will be created as a ZIP file).

The process may take several minutes to complete. After completion the ZIP file can be emailed or copied for diagnostics.
