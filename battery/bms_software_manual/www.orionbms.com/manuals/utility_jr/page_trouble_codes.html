Appendix C: Diagnostic Trouble Code Definitions

This appendix lists the various different Diagnostic Trouble Codes (DTCs / error codes) that the BMS can set as well as their common causes.

NOTE: The DTC IDs shown below in parenthesis assume that the DTC ID table has not been changed from the default values (under "CANBUS Settings" at the bottom).

Diagnostic Trouble Code List:

  • (P0A1F) Internal Commununication Fault

  • (P0A00) Internal Conversion Fault

  • (P0A80) Weak Cell Fault

  • (P0AFA) Low Cell Voltage Fault

  • (P0A04) Open Wiring Fault

  • (P0AC0) Current Sensor Fault

  • (P0A01) Pack Voltage Sensor Fault

  • (P0A02) Weak Pack Fault

  • (P0A03) Voltage Redundancy Fault

  • (P0A9C) Thermistor Fault

  • (P0A06) Charge Limit Enforcement Fault

  • (P0A07) Discharge Limit Enforcement Fault

  • (P0A08) Charger Safety Relay Fault

  • (P0A09) Internal Memory Fault

  • (P0A0A) Internal Thermistor Fault

  • (P0A0B) Internal Logic Fault
