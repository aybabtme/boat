Peak Current Limit Decrement Timer

Units: [100ms]

The BMS keeps track of how much the current limit(s) have been exceeded recently. Once the limits are no longer being exceeded (ie: both the discharge and charge current limits are being properly respected), the BMS will gradually lower these counters to 0.

This parameter represents the amount of time (in 100ms increments) that it takes for the peak current counter to return to 0. For most applications, this value can be the same as the Peak Current Limit Increment Timer.
