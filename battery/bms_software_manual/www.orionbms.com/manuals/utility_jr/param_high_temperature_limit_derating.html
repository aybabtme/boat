High Temperature Limit Derating

Units: [Amps/Celsius]

This is the number of amps-per-degree-Celsius at which the discharge current limit (DCL) / charge current limit (CCL) is reduced due to high temperature. EG: if the highest pack temperature increases by 1C, this value is the number of additional amps that the specified limit is reduced by.

This value is related to the Start Derating Limit At High Temperature parameter.

NOTE: Both the Discharge Current Limit and Charge Current Limit parameters have their own copy of this setting.
