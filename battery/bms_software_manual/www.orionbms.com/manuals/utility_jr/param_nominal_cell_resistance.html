Nominal Cell Resistance

Units: [0.01 milli-Ohms]

This is the nominal (normal) cell internal resistance at a given temperature (specifically, every 5 degrees Celsius) and provides the BMS with a starting resistance value when no historical data about the battery pack is available. This value is especially important if the application either does not have routine amperage draw (EG: a stationary application or a low current application) or does not use a current sensor.

The nominal resistance value specified is used by the BMS as a default value if the temperature of the battery pack has changed significantly enough since the last time the internal resistance was calculated. This table should be fully populated for all applications.
