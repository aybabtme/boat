Multi-Purpose Input Function

This option allows the user to specify the function of the Multi-Purpose Input (MPI) signal (pin 15 on the Main I/O harness).

Note: This input is 48v (60v max) tolerant. Do not apply a negative voltage to this input

Available Functions:

  • Ready Power Status: Using the MPI as this function allows the BMS to sense the presence of "Ready" power (ie: the application is in operation mode as opposed to charging mode). This can be useful if an application requires the presence of "Ready" power before activating the Discharge Enable output.

  • Clear Fault Codes: When this function is selected the MPI operates as a "clear error codes" input button. When raised high it will cause the BMS fault codes to be cleared.
