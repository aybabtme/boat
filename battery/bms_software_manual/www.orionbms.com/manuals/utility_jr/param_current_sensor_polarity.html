Current Sensor Polarity Inverted

Units [Enabled/Disabled]

This parameter is used to switch the polarity (directionality) of the Current Sensor. This is very helpful if the current sensor is installed backwards and the current is registering the wrong way.

NOTE: In a correctly installed system, a positive current indicates current flowing out of the battery (discharge) and a negative current indicates current flowing into the battery (charge).
