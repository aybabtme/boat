SOC Only Corrected UP On This Point

Units: [Enabled/Disabled]

If set, this value will prohibit the battery State of Charge (SOC) from correcting DOWN based on this point (it will ONLY drift up based on this point). This can be useful if a battery will routinely hit the specified voltage under load due to voltage sag and drifting is undesired.

NOTE: For details on how SOC correction / drift works, please see State of Charge Drift
