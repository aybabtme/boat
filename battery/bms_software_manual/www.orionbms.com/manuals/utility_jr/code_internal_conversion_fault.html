Internal Conversion Fault (P0A00)

This fault indicates that there was a problem detected with the internal voltage sensing circuitry.

If this error occurs, please download the associated freeze frame data and contact the factory or authorized dealer for assistance. Please save the freeze frame data and send it along with any other relevant information to the factory or authorized dealer as it may be crucial for diagnosing or repairing the unit.
