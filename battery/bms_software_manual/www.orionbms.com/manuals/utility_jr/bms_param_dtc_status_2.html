DTC Status #2

Default Length (Bytes): 2

This parameter contains the status of several BMS fault codes. The format is as follows:

Bit #1 (0x0001): P0A07 (Discharge Limit Enforcement Fault)
Bit #2 (0x0002): P0A08 (Charger Safety Relay Fault)
Bit #3 (0x0004): P0A09 (Internal Memory Fault)
Bit #4 (0x0008): P0A0A (Internal Thermistor Fault)
Bit #5 (0x0010): P0A0B (Internal Logic Fault)
Bit #6 (0x0020): RESERVED
Bit #7 (0x0040): RESERVED
Bit #8 (0x0080): RESERVED

Bit #9 (0x0100): RESERVED
Bit #10 (0x0200): RESERVED
Bit #11 (0x0400): RESERVED
Bit #12 (0x0800): RESERVED
Bit #13 (0x1000): RESERVED
Bit #14 (0x2000): RESERVED
Bit #15 (0x4000): RESERVED
Bit #16 (0x8000): RESERVED

For all of these values, a "1' signifies that the signal or relay in question is enabled or active (high). A "0' indicates that it is inactive or off (low).
