Voltage Restore Limit Timer

Units: [100ms]

This parameter is the number of 100ms intervals that must pass (when the voltage limit is NOT exceeded) before the associated battery current limit is restored by 10% of the maximum limit.

This timer can be adjusted to avoid oscillations in the current limits. A value of 0 indicates that the associated limit (CCL / DCL) will immediately be restored to normal once the voltage maximum is no longer being exceeded.

This is also the timer that is used to restore limits to their original values following a voltage sensor failure condition (see: Voltage Fault Derate Timer).
