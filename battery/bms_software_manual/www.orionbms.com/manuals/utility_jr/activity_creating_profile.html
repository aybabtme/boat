Creating and Uploading a Profile

  1. Open the OrionBMS utility.

  2. Connect to the BMS.

  3. Go through the Profile Setup Wizard to generate the battery profile settings.

  4. Press the "Transmit All Settings" button on the Battery Profile screen to transmit the newly created battery profile to the BMS.

The existing battery profile on the BMS can be retreived by pressing the "Receive All Settings" button while connected to the BMS.

NOTE: Receiving the existing profile from the BMS will override all current profile settings in the program. Be sure to save your current profile before retreiving the profile from the BMS.
