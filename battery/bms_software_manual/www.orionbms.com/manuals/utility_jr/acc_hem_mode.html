Hybrid Energy Manager Modes

These modes require the Hybrid Energy Manager (HEM). There are currently several supported modes for the HEM:

  1. Hybrid Mode: This reverts the vehicle back to normal factory control.

  2. PHEV Mode: This mode will use the available electric power of the vehicle as aggressively as possible (the maximum ratings) however the gasoline engine is still active to provide additional power. It is designed to offer the maximum electric power of the vehicle without restricting the total vehicle power.

  3. EV Mode: This mode will exclusively use the electric drive system to power the vehicle. During this mode, no gasoline is burned.

  4. Glide Mode: This mode will allow the gasoline engine to shut off more regularly and is ideal for short trips.
