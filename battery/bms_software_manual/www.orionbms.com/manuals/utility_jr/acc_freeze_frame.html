Freeze Frame Data

Freeze frame data is a collection of parameters and data that are recorded in memory when a Diagnostic Trouble Code (DTC) is set. Typically an ECU will support at least 1 or 2 freeze frame buffers.

The collected data is typically accessed via an OBD2 interface. The OrionBMS supports accessing freeze frame and diagnostic trouble codes through the utility application provided with it.
