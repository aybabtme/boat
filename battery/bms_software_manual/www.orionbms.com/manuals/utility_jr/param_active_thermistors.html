Active Thermistors

Units: [Enabled/Disabled]

These boxes represent which thermistors have been populated on the BMS wiring harness. If a thermistor is not installed, the respective box should be unchecked to prevent an error condition from being set. The BMS will ignore non-populated (unchecked) thermistors.

The BMS has support for up to three (3) thermistors on the main unit (NOTE: Rev-B or older hardware only supports 2 thermistors). It can support additional thermistors through an external thermistor expansion module. For more information please visit www.orionbms.com.
