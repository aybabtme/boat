Peak Current Limit Increment Timer

Units: [100ms]

This parameter represents the amount of time (in 100ms increments) that the Peak Current Limit is allowed to be sustained before a code is set.

Example: If the maximum discharge limit is set to 100 Amps and the Peak Current Limit Percentage is set to 20%, then the Peak Current Limit would be 120 amps (100 amps + 20% of 100). If the Peak Current Limit Timer (this parameter) is set to 5 seconds, then the BMS would permit a discharge of 120 amps for 5 seconds before a code would be set.

Following this example, if the discharge amperage was observed to be 140 amps, the BMS would weigh this differently and it would throw an error code sooner.
