On-Board Diagnostics II (OBD2)

The On-Board Diagnostics II (commonly referred to as OBD2) protocol is used primarily in automotive applications as a standardized means to retreive real-time information from a control device as well as poll diagnostic trouble information. OBD2 allows for a uniformal way of accessing data from this device regardless of what application it is being used in.

This particular device uses the CANBUS portion of the OBD2 standard.
